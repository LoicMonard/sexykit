Select component is perfect to display plenty of options

Basic usage

The default Select component is a simple input with a dropdown list. It works with any kind of data, as long as you provide the right keys to access the desired value and label . The Select component also includes a search input to filter the list of options.

Sizes

As the Select component is based on the Textfield component, it supports the same sizes. It can also be used with an icon with the icon property.

Data binding

To add two way data binding over the Select component, simply pass your reference as the value property, and make sure to update your reference whenever the Select change event is fired.

Disabled select

To disable the Select component, simply pass the disabled property.

Disabled options

To disable a specific option, simply pass the disabled property to the option object.
